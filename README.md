\# Search Engine C++



Финальный проект курса \*\*«Разработчик на C++ с нуля»\*\*.



Проект реализует простой поисковый движок по текстовым документам.



\## Возможности



\- Индексация текстовых файлов

\- Поиск документов по словам

\- Расчёт релевантности результатов

\- Сортировка по релевантности

\- Ограничение количества результатов

\- Конфигурация через JSON

\- Многопоточная индексация документов



\---



\# Стек технологий



\- C++

\- STL

\- JSON (nlohmann/json)

\- CMake

\- Multithreading (std::thread)
